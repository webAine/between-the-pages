# Геймдизайн — Between the Pages

## Основной цикл

```
Начало дня → Приходит покупатель → Определяешь запрос → Ищешь в каталоге → Ищешь на полке → Продаёшь / Предлагаешь альтернативу → Следующий покупатель → Конец дня
```

---

## Покупатели

### Типы запросов

| Тип | Что говорит | Как искать | Риск |
|-----|-------------|------------|------|
| Знает название | «Мне нужна "Имя книги"» | Каталог → название → физические признаки | Низкий |
| Знает примерно | «Что-то про море... детектив, кажется» | Каталог → жанр → несколько вариантов | Средний |
| Не знает | Молчит, смотрит, оговаривается | Читаешь покупателя → интуиция | Высокий |

### Чтение покупателя (тип «не знает»)
Визуальные подсказки + диалоговые реплики дают намёки на жанр/настроение:
- Усталый вид + упоминание «давно не читал» → лёгкое, ненапрягающее
- Нервничает, смотрит на часы → короткое, захватывающее
- Пришёл с ребёнком → детская / семейная
- Красные глаза → что-то про потерю, принятие

## Каталог

Физическая книга на столе. Навигация по жанровым закладкам.

### Карточка книги
```
Название: ...
Автор: ...
Жанр: ...
─────────────────
Описание: (2-3 предложения сюжета)
─────────────────
Физические признаки:
  Обложка: [цвет] [тип — твёрдая/мягкая]
  Толщина: [тонкая / средняя / толстая]
  Где должна стоять: Секция [жанр], полка [верхняя/средняя/нижняя], примерно [начало/середина/конец]
```

---

## Планировка экрана

Вид сбоку. Один экран — одна «зона». Основных зон две:

**Зона магазина (основная)**
```
┌─────────────────────────────────────────────────────┐
│  [Покупатель у прилавка]          [Прилавок + касса] │
│                                                       │
│  [Шкаф 1] [Шкаф 2] [Шкаф 3] ──→ (прокрутка влево)  │
│                                                       │
│  [Каталог на столе — кликабельный]                   │
└─────────────────────────────────────────────────────┘
```

- Шкафы прокручиваются горизонтально (пан влево), если их больше одного на экране
- Каталог — отдельный объект на столе, клик открывает UI поверх
- Покупатель стоит у прилавка пока ждёт

**Скролл полок**
- Видно 1–2 шкафа одновременно
- Стрелка/свайп влево → следующий шкаф
- Каждый шкаф = один жанровый раздел (или два в хаосе)
- Позиция шкафа запоминается между поисками

**Темп**
- Таймера нет, покупатель ждёт без давления
- Напряжение создаётся репутационным штрафом за долгое ожидание, а не таймером

## Полки

### Состояние полок
- **Беспорядок** (старт) — книги стоят непоследовательно, надо искать по физическим признакам
- **Частичный порядок** (после каталогизации) — жанры разделены, но внутри жанра ещё хаос
- **Порядок** (прогрессия) — секции и полки соответствуют каталогу, поиск быстрый

### Механика поиска
1. Получаешь физические признаки из карточки каталога
2. Идёшь к нужной секции (или туда, где примерно должна быть)
3. Просматриваешь книги по цвету обложки и толщине
4. Находишь → берёшь → несёшь покупателю

Если не нашёл:
- Ищешь дольше (расход времени)
- Предлагаешь замену из каталога
- Извиняешься («нет в наличии») — штраф к репутации

### Время поиска зависит от порядка
| Уровень порядка | Время поиска |
|-----------------|--------------|
| Хаос            | ~3 минуты    |
| Частичный       | ~1.5 минуты  |
| Порядок         | ~30 секунд   |

## Ресурсы

### Деньги
- Приходят от продаж
- Расходы: аренда (еженедельно), закупка новых книг
- Банкротство = конец игры (мягкий, с возможностью отыграть день)

### Репутация
- Шкала 0–100, старт ~30
- +5 за точное попадание (покупатель доволен)
- +2 за хорошую альтернативу
- -3 за неверный подбор
- -5 за долгое ожидание
- -10 за «нет в наличии» без замены
- 0 репутации → перестают приходить постоянные покупатели

### Порядок на полках
- Шкала 0–100, старт ~10
- Растёт от ручной каталогизации (действие «разобрать полку»)
- Падает в дни наплыва покупателей (берёшь книги, не всегда ставишь обратно)
- 0 → невозможно найти что-либо быстро

## Прогрессия магазина

### День 1–7: Хаос
- Всё перемешано
- Постоянные покупатели холодные, недоверчивые
- Первые страницы дневника — в случайных книгах

### День 8–21: Порядок наводится
- Начинают появляться постоянные покупатели
- Каталог изучен лучше → поиск ускоряется
- Антагонист появляется лично

### День 22+: Магазин работает
- Репутация высокая → приходят особые покупатели
- Редкие книги открывают новые воспоминания деда
- Финальный акт истории

---

## Действия между покупателями

- **Разобрать полку** — тратишь время дня, порядок +X
- **Просмотреть каталог** — учишь расположение книг (пассивное ускорение поиска)
- **Закупить книги** — тратишь деньги, пополняешь ассортимент
- **Прочитать страницу дневника** — если нашёл сегодня

---

## Конец дня

Короткий экран итогов:
- Деньги: приход / расход / баланс
- Репутация: изменение
- Порядок: изменение
- Найдено страниц дневника: N
